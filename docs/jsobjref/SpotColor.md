# SpotColor

`new SpotColor()`

#### Description

Color class used to apply the color value of a spot at a specified tint value. Can be used in any property that takes a [Color](./Color.md) object.

---

## Properties

### SpotColor.spot

`spotColor.spot`

#### Description

A reference to the spot color object that defines the color.

#### Type

[Spot](./Spot.md)

---

### SpotColor.tint

`spotColor.tint`

#### Description

The tint of the color.

Range: 0.0 to 100.0

#### Type

Number (double)

---

### SpotColor.typename

`spotColor.typename`

#### Description

The class name of the referenced object.

#### Type

String; read-only.
